公司简介

 上海首丰精密仪器有限公司成立于2005年，坐落于上海市市区交通枢纽中心，地处南外滩南浦大桥附近。公司专业代理日本三丰（Mitutoyo）全系列产品，主要产品包括如下：千分尺、微分头、内径千分尺、卡尺、高度尺、高度仪、深度尺、指针式指示表 杠杆指示表、量块校准仪器、精密传感器、Digimatic标尺和光栅尺、投影仪、显微镜、形状测量仪、影像测量仪、三坐标测量机、硬度试验机、厚度表等。公司自成立以来一直秉承三丰公司「品质第一」的宗旨，为全国不断扩大的市场提供各类仪器设备及其售后服务。
